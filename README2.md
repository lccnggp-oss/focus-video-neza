# Galería Estática para Fotógrafo – Focus Dron

Sistema 100% estático para entregar galerías privadas a clientes. Cada cliente recibe una carpeta con acceso por código, selección de fotos para book y descarga en ZIP.

## Cómo usar

1. **Copia la plantilla**  
   Copia la carpeta `template-cliente` dentro de `clientes/` con el nombre del evento:
   ```bash
   cp -r template-cliente clientes/boda-juan-ana
